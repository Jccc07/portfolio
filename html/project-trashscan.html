TrashScan

A Surveillance-Powered Integrated System for Monitoring Real-Time Street Litter Density utilizing Convolutional Neural Networks

Computer Vision Flask/Python Electron/Node.js

Project Description

TrashScan is an innovative surveillance-powered integrated system designed to monitor real-time street litter density using advanced Convolutional Neural Networks (CNN). This system leverages computer vision technology to detect and classify different types of litter in urban environments, providing municipal authorities with actionable data for more efficient waste management and street cleaning operations.

The application combines a robust Flask/Python backend for AI processing with an Electron/Node.js desktop interface, creating a comprehensive solution for smart city waste management. The system can identify various types of litter, track accumulation patterns, and generate heat maps of high-density litter areas to optimize cleaning routes and resource allocation.
