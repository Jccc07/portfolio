VYPR

A Language That Makes It Simpler and Runs Anywhere

C++

Project Description

Vypr offers a clean syntax inspired by a combination of Python and Javascript, but with some extra unique features for readability and simplicity. This source code is an implementation of a compiler that generates IR code, translates it into Python code, and offers a way to run and interact with Vypr code.

In summary, Vypr provides a user-friendly programming language with enhanced readability and simplicity, supported by a custom compiler that bridges Vypr code to Python for execution. This approach makes it easier for users to write, run, and experiment with Vypr code seamlessly.
